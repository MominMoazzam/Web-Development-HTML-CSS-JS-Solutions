Mostly used Input tag with different type's value.

Note: An Input tag having default value of type attribute is text.

Type : text

An input tag where anything can be written.

Type : email

An input tag used for email

Type : password

An input tag used to write password which is hidden by default.

Type : number

An input tag which accepts only Number, also can made more meaningfull by adding min and max attributes.

Type : radio

An input tag has a great use when have to select single thing from many thing.

Type : checkbox

An input tag has a great use when have to select multiple thing.

Type : file

An input tag to select any file from your local machine's.

Type : date

An input tag, which provides a calender to select the date from it.

Type : reset

An input tag to display a button, upon clicking it clear all the filled forms fields.

Type : button

An input tag to display a simple button.

Type : submit

An input tag to display a button, upon clicking it submit the forms.
